## JAVA排序算法
#### 1.冒泡排序
```java
    public static int[] bubbleSort(int[] arr) {
        for (int i=0; i<arr.length-1; i++) {
            for (int j=i+1; j<arr.length; j++) {
                if (arr[i] > arr[j]) {
                    int temp = arr[i];
                    arr[i] = arr[j];
                    arr[j] = temp;
                }
            }
        }
        return arr;
    }
```
#### 2.选择排序
```java
    public static int[] selectionSort(int[] arr) {
        for (int i=0;i<arr.length-1; i++) {
            int index = i;
            for (int j=i+1; j<arr.length; j++) {
                if (arr[i] > arr[j]) index = j;
            }
            int temp = arr[i];
            arr[i] = arr[index];
            arr[index] = temp;
        }
        return arr;
    }
```
